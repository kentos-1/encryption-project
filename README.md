# Криптографический инструмент

Курсовая работа по программированию

## Описание
Приложение для шифрования и дешифрования файлов с графическим интерфейсом.
Использует гибридную архитектуру: 
- **C++** - криптографическое ядро (AES-256)
- **Python** - графический интерфейс (Tkinter)

## Функциональность
- Шифрование файлов с паролем
- Дешифрование файлов
- Индикатор сложности пароля
- Прогресс-бар
- Кроссплатформенность (macOS, Linux, Windows)

## Установка и запуск

### Требования
- Python 3.11+
- Компилятор C++ (g++/clang)
- make

### Сборка
```bash
# Компиляция C++ библиотеки
cd crypto_lib
make
make install
cd ..
# encryption-project
